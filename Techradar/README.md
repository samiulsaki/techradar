# Fluxus Logicae Tech and Tools Radar - Content

This is the location of Fluxus Logicae Tech and Tools Radar content - published (locally) under: https://localhost:3000/fluxus-logicae-techradar/

If you want to build your own techradar you may want to have a look at https://github.com/AOEpeople/aoe_technology_radar instead.

## Content Guidelines

New blips should be tagged. Tags are used like this:

```md
tags: [AUTOMATION, PLATFORM-ENGINEERING]]
```

## Development

### Build the radar

```
npm i
npm run serve
```

Then open here: http://localhost:3000/fluxus-logicae-techradar

### Build the radar with static files

```
npm i
npm run build
```
